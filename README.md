# Основная логика
Расписание с фильтрацией и возможность зарегистироваться:
- Просмотр всей проги
- На определённый день
- В определённом корпусе
Регистрация по определённой ссылке для пиара, складывать в табличку
Отметка с динамичным кьюаром (После каждой отметки изображение в сообщении изменяется, чтобы ссылку нельзя было слить)

Отправка письма:
Нужен человек, который будет объяснять, что делать и проверять, что человек присвоил получателя в боте
1. Сканирование кьюара на пустом письме (В разных корпусах разные кьюары, указывающие на почтовый пункт)
2. Выбор корпуса для отправки
3. Выбор получателя
4. Подтверждение ввода (Айди письма связан с данными о нём)
5. Написание письма
6. Отправка в ящик

Сортировка:
1. Достать письмо
2. Отсканировать кьюар, увидеть данные о нём
3. Отложить в нужную стопку - корпус
4. Добавить в таблицу айди письма (можно автоматически при сканировании организатором)

Транспортировка:
1. Взять стопку
2. Отнести в нужный корпус
3. Нажать на кнопку, которая сделает рассылку по людям из стопки (Письмо с номером ... пришло в корпус ...)
4. Человек приходит
5. Организатор находит письмо с нужным номером (как это ускорить?) и отмечает, что оно получено — приходит уведомление отправителю и закрывается "транзакция"
6. Выдаёт человеку
